## 📊 Key Movements

**LangGraph** holds the top spot with the strongest combination of GitHub activity (34.5k stars) and production adoption. Its graph-based architecture continues to attract enterprise teams building complex agent workflows.

**CrewAI** maintains #2 with explosive community growth (25.4k stars) and the most accessible multi-agent API. The role-based design pattern resonates with teams new to agent development.

**AutoGen** (Microsoft) rounds out the top 3. Its conversational agent patterns and strong Microsoft ecosystem integration make it the default choice for Azure-heavy shops.

**OpenAI Agents SDK** enters at #4 — newer but backed by OpenAI's reach. Watch this space as their Responses API matures.

**Semantic Kernel** holds at #5, the enterprise workhorse with deep .NET/Java support that larger organizations need.

## 🔍 Framework to Watch

**smolagents** (HuggingFace) — lightweight, code-first agents. Not yet in the top 5 but growing fast with the open-source ML community. If you're building agents around open models, keep an eye on this one.
