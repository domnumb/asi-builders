# LinkedIn Post — W12 (March 21, 2026)

## Option A — IBM's Bet on AI Agents (Tech Leadership angle)

🏗️ IBM just mass-deployed autonomous agents across its consulting division.

Not copilots. Not chatbots. Full-loop agents that:
→ Read client briefs
→ Draft deliverables
→ Route to human review only when confidence drops below threshold

The shift is clear: enterprises aren't asking "should we use AI?" anymore.
They're asking "how many humans do we keep in the loop?"

I've been tracking this at ASI Builders — the companies moving fastest share 3 patterns:

1. They treat agents as teammates, not tools (task ownership, not task assistance)
2. They measure agent output the same way they measure human output
3. They invest in agent observability before scaling

The gap between "AI-curious" and "AI-native" orgs is widening every week.

## Option B — Video Gen Wars: Wan 2.1 vs Veo 3 (Builder angle)

🎬 I benchmarked Wan 2.1 against Veo 3 this week.

The result? Neither wins outright — but the gap tells you where the market is heading.

Wan 2.1 (open-source, Alibaba):
✅ Better motion coherence on long sequences
✅ Runs locally — no API costs
❌ Struggles with photorealistic faces

Veo 3 (Google DeepMind):
✅ Stunning photorealism
✅ Better prompt adherence
❌ Locked behind API, expensive at scale

The real insight: open-source video gen is ~6 months behind closed models.
That gap was 18 months a year ago.

For builders: if you're prototyping video workflows, start with Wan locally.
If you're shipping to clients, Veo 3 is worth the cost.
